## ğŸ§± System Architecture (High-Level)
PHR Vault follows five core layers:

1. **Client Layer**  
   - React/Next.js front-end  
   - Connect wallet, manage records, share permissions  

2. **Smart Contract Layer**  
   - Solidity contracts for data access control  
   - Role-based permissions  
   - Event logging for audits  

3. **Decentralized Storage Layer**  
   - IPFS for secure document storage  
   - Content addressing for integrity  

4. **Identity & Access Layer**  
   - Public/private key ownership  
   - Zero-trust authentication model  

5. **Off-Chain Metadata Layer**  
   - Record indexing  
   - UX and performance improvements
